# ☁ Cloud-Based File Storage and Sharing System

## 📌 Project Overview

The Cloud-Based File Storage and Sharing System is a web application that allows users to securely upload, store, manage, and share files using cloud technology.

This project is developed as a mini cloud computing project using HTML, CSS, Python Flask, MySQL, and AWS S3.

---

## 🎯 Objectives

- Provide secure cloud-based file storage.
- Allow users to upload and manage files.
- Enable file sharing and downloading.
- Store file information in a database.
- Demonstrate cloud computing concepts using AWS.

---

## 🚀 Features

### User Module
- User Registration
- User Login

### File Management
- Upload Files
- View Uploaded Files

### Cloud Integration
